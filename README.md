# topo-lang.github.io

Topo organization landing page — deployed at [topo-lang.github.io](https://topo-lang.github.io).

## Development

This is a static HTML page. Edit `index.html` directly. No build step.

## Custom domain

To add a custom domain, create a `CNAME` file in this repo root with the domain name. GitHub Pages will handle HTTPS via Let's Encrypt.
